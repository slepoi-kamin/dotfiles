# Config Patterns

> How each config file is structured.

## `.zshrc` (Zsh Configuration)

```bash
# Sections (top to bottom):
1. Kiro CLI pre-block (keep at top)
2. Powerlevel10k instant prompt
3. Homebrew path setup
4. zinit plugin manager setup
5. History configuration
6. zinit plugin loading (powerlevel10k, zsh-*)
7. Completions setup
8. Keybindings
9. Completion styling
10. Aliases
11. Shell integrations (fzf, zoxide)
12. Custom scripts
13. Kiro CLI post-block (keep at bottom)
```

### Key Patterns

```bash
# zinit plugin loading
zinit light zsh-users/zsh-syntax-highlighting

# zinit snippet (Oh My Zsh)
zinit snippet OMZP::git

# Conditional source (optional files)
[[ -f ~/.p10k.zsh ]] && source ~/.p10k.zsh
```

---

## `.tmux.conf` (Tmux Configuration)

- Tmux plugin manager (tpm) plugins
- Catppuccin theme
- Prefix key: `Ctrl+f`
- Split keys: `|` (vertical), `-` (horizontal)

---

## `.alacritty.toml` (Alacritty Config)

- TOML format
- Font: JetBrains Mono Nerd Font
- Theme: Custom (not Catppuccin)

---

## `.p10k.zsh` (Powerlevel10k)

- Generated by `p10k configure`
- Instant prompt enabled
- Custom colors and icons

---

## Config Loading Order

```
~/.zshrc → sources ~/.p10k.zsh
~/.tmux.conf → loaded by tmux
~/.alacritty.toml → loaded by Alacritty
```

---

## Custom Scripts

Scripts in `scripts/zsh/*.zsh` are sourced by `.zshrc`:

```bash
ZSH_SCRIPTS=${${(%):-%N}:A:h}/scripts/zsh
for file in $ZSH_SCRIPTS/*.zsh; do
  source "$file"
done
```
